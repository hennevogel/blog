  • Customer: openSUSE, GNOME, ownCloud
  • Github: openSUSE/osem
  • Online: osem.io, Demo Instance

Open Build Service - Software Release Engineering System

Helps Free Software developers with the compilation, assembly, and delivery of source code into finished products for a wide range of Linux operating systems and hardware architectures. I am responsible for the Ruby on Rails backend and frontend (UI/API). I also contribute to many other aspects of this project. Like developer relations & marketing, writing documentation, operations of the reference server etc.

  • Role: Lead Developer, Operations
  • Platform: Web
  • Technology: Ruby on Rails, Bootstrap, SQL, rspec, OCI Container
  • Customer: SUSE, openSUSE, Linux Foundation, Tizen, Intel
  • Github: openSUSE/open-build-service
  • Online: openbuildservice.org, Reference Server

influxdb-rails - Application Performance Metrics

A RubyGem that helps people to track performance and health of Ruby on Rails applications. By writing metrics into a time series database (InfluxDB) and providing dashboards (Grafana) to visualize those metrics.

  • Roles: Lead Developer
  • Technology: Ruby, InfluxDB, Grafana
  • Customer: SUSE, openSUSE
  • Github: influxdata/influxdb-rails
  • Online: Rubygems, Reference Dashboards

openSUSE - Linux Operating System

A stable, easy to use and multi-purpose Linux based operating system. Helps people to get all their of computing done, from browsing facebook to developing complex applications. I was part of the group of engineers that launched this community in 2005 and I'm still a member with various roles today. I maintain software packages that are included in the OS, I operate several servers/applications that the project utilizes and I contribute to the promotion by writing, speaking and evangelizing about it.

  • Roles: Developer, Operation, Developer Relations, Editor
  • Technology: Linux, Apache, Postfix, Wordpress, Mediawiki, Open Build Service, OSEM
  • Online: opensuse.org
